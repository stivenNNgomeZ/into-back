# into-back - Memoria del Proyecto

## Estado actual
- Proyecto backend Node.js + Express para el SENA (Técnico en programación de software)
- Repositorio: https://github.com/stivenNNgomeZ/into-back

## Actividades completadas

### Actividad de Apropiación ✅
- Servidor básico en puerto 3000
- Rutas: `GET /`, `GET /aprendices`, `GET /programas`
- Evidencias del servidor funcionando en `evidencias/`

### Actividad de Transferencia ✅
- Estructura modular con `src/routes/`
- Rutas de usuarios: `GET /users`, `POST /users`
- Rutas de tareas: `GET /tasks`, `POST /tasks`
- README con propósito, estructura y cómo ejecutar
- Por ahora solo mensajes de texto, sin base de datos

## Próximos pasos (guía futura)
Según la guía, el proyecto seguirá creciendo con:
- Controladores
- Middleware
- Conexión a base de datos
- Autenticación
- Validaciones
- Integración con frontend

## Cómo iniciar
```bash
npm start
```
